# Milestone T - Evidence Exports

Goal: make sample/video validation produce artifacts that clearly show what the app generated.

## Tasks

- [ ] Export a side-by-side or split processed MP4 for selected videos.
- [ ] Include source name, selected mode, view mode, band, amplification, frame count, and quality summary in metadata.
- [ ] Add an optional signal/quality timeline artifact for processed videos.
- [ ] Make recent exports easy to share from the app.
- [ ] Add tests for export metadata and frame-processing consistency.

## Done When

- A selected sample video produces inspectable media plus metadata without relying on live phone observation.
- The exported artifact makes Raw vs processed behavior clear.
- Tests and docs are committed and pushed to `main`.
